Plugin icon for Edit → Plugins browser.

1. Save your PNG as a square image (128×128 pixels recommended).
2. Name it exactly:  Icon128.png
3. Place it in this folder (next to this README).

Path in the plugin:
  Anim8Pipeline/Resources/Icon128.png

Unreal picks this up automatically when the plugin loads.
Restart the editor after adding or changing the icon.
